Olá, obrigado por usar o WWC

siga o passo a passo abaixo para instalar o WWC no seu site

1 - Coloque todo o conteudo no seu servidor (recomendo o uso de uma pasta para colocar o conteudo)
2 - Crie uma database vazia no MySQL
3 - No seu browser, abra a pasta instalar, ex: www.meusite.com.br/webchat/instalar
4 - Preencha os campos do formulario (na database, coloque a database vazia previamente criada)
5 - Quando a instalação estiver concluida, lembre-se de apagar a pasta "instalar" do servidor

Para gerenciar as salas, pelo browser entre na pasta "admin" e digite a senha de gerenciamento
essa senha só pode ser mudada diretamente pelo MySQL

Para quem tiver um conhecimento intermediário de PHP e MySQL, é possivel alterar as emoticons:
é só associar a tabela emoticons com as imagens contidas na pasta emoticons

Para obter todos os comandos possiveis do WWC, digita /ajuda em qualquer sala de conversa

Para ligar o robot, use um brower e mantenha a página robot.php aberta
A função do robot é enviar a cada 5 minutos uma mensagem da tabela auto_msg
